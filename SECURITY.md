# Security Policy

## Supported Versions

Currently, only the latest version of this personal website is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it privately. 

**Do not open a public issue.**

Please send an email to [adarshan20302@gmail.com](mailto:adarshan20302@gmail.com) with the details of the vulnerability. I will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.

Thank you for helping keep this project secure!

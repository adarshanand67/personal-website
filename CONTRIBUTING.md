# Contributing to Adarsh's Personal Website

First off, thanks for taking the time to contribute! This project is a personal portfolio, but I welcome feedback, bug fixes, and suggestions.

## How Can I Contribute?

### Reporting Bugs

- Use the GitHub issue tracker.
- Provide a clear description of the problem and steps to reproduce.

### Suggesting Enhancements

- Open an issue to discuss the enhancement first.
- Focus on performance, accessibility (A11y), or UI/UX improvements.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. Ensure the project builds (`bun run build`).
3. Run existing tests (`bun test`).
4. If you've added code that should be tested, add tests.
5. Ensure your code follows the existing style (JSDoc, TypeScript).

## Local Development

Requires [Bun.js](https://bun.sh/).

```bash
bun install
bun dev
```
